GIT cheatsheet

最近又开始捣鼓这个闲置多年的博客。期间用 git 出现了些问题。平时用不到 git 重新简单学习了下，发现很好玩。整理了常用命令，以备后续查看。 后面的维护，会尽量尝试 git 工具和技巧，也会添加新内容进来。

参考 GIT-BOOK

帮助命令 git xxx -h

GIT 初始化
~ git init
Status
~ git status -s
| ??  untracked_file.md
| A_  staged_new_file.md
| _M  untracked_modified.md
| _M  staged_modified.md
| MM  staged_and_unstaged_modified.md
| └╂───────staged
|  ┗━━━━━━━unstaged
Add & Commit & Restore
~ git add content/post/git_cheatsheet.d
~ git add -A
~ git commit -m "commit message"
~ git restore [file]
~ git restore --staged [file]
~ git commit -m "new post"
~ git commit --amend
Diff
~ git diff                  # unstaged and staged
~ git diff file             # compare file
~ git diff --staged         # staged and commited 
~ git diff branch-name      # HEAD and branch
~ git diff origin/main      # HEAD and origin/main
~ git diff HEAD~1           # HEAD and previous commit
~ git diff branch01 branch02 # branch and branch
~ git difftool
Log & Show
~ git log
~ git log --stat
~ git log --oneline
~ git log --graph --oneline --decorate --all  
# branch graphic

~ git show c557fdf
~ git show HEAD~1
~ git show HEAD~1:filename
Remote
~ git remote add origin https://github.com/xxx/xxx.git  
# add new remote named as origin
~ git clone https://github.com/xxx/xxx.git      
# copy from server
~ git fetch origin main     
# fetch from remote [origin] branch [main]
~ git pull origin main      
# fetch and merge from remote [origin] branch [main]
~ git push origin main      
# push local to remote [origin] branch [main]
Branch
~ git branch --all          # show new branch
~ git branch new-branch     # create new branch
~ git switch new-branch     # switch to brance
~ git branch -d new-branch  # delete branch
~ git push -u origin new-branch # push branch to remote 
~ git switch main
~ git merge branch          # mergin branch to main
~ git mergetool
Config
~ git config --global -e    # 打开 config
~ git config --global core.editor "code --wait"
~ git config --global merge.tool vscode
~ git config --global mergetool.vscode.cmd "code --wait $MERGED"
~ git config --global diff.tool vscode
~ git config --global difftool.vscode.cmd "code --wait --diff $LOCAL $REMOTE"

~ git config --global core.autocrlf true    
# for win: lf>crlf when pull; crlf>lf when push;

~ git config --global core.autocrlf input   
# for mac and linux: crlf>lf when pull;
