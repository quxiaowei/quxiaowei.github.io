ffmpeg cheatsheet

2016-07-15

-vcodec -acodec 视屏，音频编码 copy 拷贝源文件

合并视屏 ffmpeg -f concat -i buffer_list -c copy output.mkv

截取视频 ffmpeg -i source.avi -ss 00:01:35 -t 00:47:13 -vcodec copy output.mp4 -ss 开始时间 -t 时长

缩小视屏 ffmpeg -i a.mp4 -vf scale=853:480 -acodec acc -vcodec h264 out.mp4

裁剪视屏 ffmpeg -i a.mp4 -strict -2 -vf crop=1080:1080:0:420 out.mp4 crop=width:height:x:y
